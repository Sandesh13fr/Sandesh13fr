## 👨‍💻 About Me

I'm a **Full-Stack Developer** and **AI/ML enthusiast** from India, passionate about building things at the intersection of code, design, and real-world impact.

- 🔬 Building **[SpeciesBench](https://github.com/Sandesh13fr/speciesist-bias-benchmark)** — a production-grade AI bias benchmarking framework for [OpenPAWS](https://openpaws.ai)
- 🔍 Created **[FindXVision](https://github.com/Sandesh13fr/FindXVision)** — a MERN-stack missing person platform with face recognition & real-time alerts
- 🤖 Built **[Baymax-AI](https://baymax-v13.streamlit.app/)** — an LLM-powered healthcare companion (Groq + Streamlit)
- ✍️ Writing on [Medium](https://medium.com/@sandeshdawkhar13) & [Dev.to](https://dev.to/sandesh13fr)
- ⚡ *I believe in constant improvement — each line of code is a step closer to mastery!*
